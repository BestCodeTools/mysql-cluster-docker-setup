# PoC Python + MySQL Cluster

Esta PoC valida:

1. migration com `Alembic`
2. CRUD com SQLAlchemy Core
3. CRUD com SQLAlchemy ORM

## Uso

```bash
cd poc-python-cluster
bash ./run-poc-python.sh
```

## Variaveis

```bash
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=cluster_app
DB_PASSWORD=ClusterApp123!
DB_NAME=cluster_poc
```
