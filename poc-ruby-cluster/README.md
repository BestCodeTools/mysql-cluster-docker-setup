# PoC Ruby + MySQL Cluster

Esta PoC valida:

1. migration com `ActiveRecord`
2. CRUD com raw query
3. CRUD com `ActiveRecord`

## Uso

```bash
cd poc-ruby-cluster
bash ./run-poc-ruby.sh
```

## Variaveis

```bash
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=cluster_app
DB_PASSWORD=ClusterApp123!
DB_NAME=cluster_poc
```
